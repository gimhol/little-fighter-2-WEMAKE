# [RN-LF2:Wemake] 开工预告

经过一上午的摸鱼（年前尤甚……），决定还是先放下LF2:Wemake其他功能性的开发，
专注到数据转换功能与逻辑的正确性上。

毕竟再怎么弄，对于玩家来说都不是新内容。

而我觉得大多数玩家还是需要新内容（或者是没尝试过的内容）才能吸引来的。

原谅我擅自找了RN-LF2来验证下LF2:Wemake是否能正确转换全部原版的改档数据。

此处@RN-LF2的原作者

================================================================================

目前碰到问题：

一、Windows下对大小写并不敏感，给文件改了下名就能继续施工了。

二、粗略的尝试下，转换后的角色倒是基本能正常使用

    1.“天上漂浮的Template”存在不能转向和攻击的问题

    2.Felix空中飞刀，落地不会消失……

    3.其他还没仔细看

    4.发现没有Bat，我确实也改考虑同ID角色的问题了，LF2:Wemake不像LF2，不允许OID重复。

三、闯关模式……

    这肯定涉及我当年不知道的黑科技，所以一点都玩不了。

    过去我其实从来就没“精通”过改档，很多东西都是一知半解的。

    1-1，黑幕倒是能正常播放，然后什么反应都没有，游戏就结束了。

    1-2，“对话框”里Bandit头像位置也不对，然后我就绝望的没继续试下去了。

    后面暂时懒得一关关的看了，问题估计是海量的。

    四、RN-LF2的字体太特殊了……而且是画在图片上的。

    LF2:Wemake很多文字渲染都不再是直接渲染图片的了，所以目前还支持不了。

================================================================================
